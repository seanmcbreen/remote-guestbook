Amazing Demo

Open in Open in Cloud Code

What Happened

With a single click [well set of them] I can:

  • Open Visual Studio Code
  • Clone a specified Git repo
  • Have that recommend 'remote development'
  • Have that configure the 'remote machine'
  • Start working...

How did it Happen

VS Code has a feature called 'remote development' which allows the front end to connect to a remote back-end for all the tools engine tasks. For instance this covers:
  • File system
  • Extensions
  • Code Indexing
  • Language Services
  • Terminal
  • Debuggers
  • Developer Tool Chain
The remote machine can either be pre-configured and you SSH into it e.g. Cloud Shell. Or you can dynamically create a Docker image (or download one). In this case I used a docker image which installs:
  • Latest node:lts image
  • The cloud code and docker extensions
  • Adds npm, typescript and tslint
  • Installs git
  • installs Docker CE
  • installs GCloud and kubectl
  • Installs skaffold
  • Installs helm
... as a result the user simply connects to GCP (via gcloud init) and everything just works.
